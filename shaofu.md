# 少妇战法（BBIKDJSelector）的主要业务逻辑
基于BBI（多空指标）、KDJ和MACD指标的组合筛选策略，具体如下：

1.BBI指标判断
   - 计算BBI线（3日、6日、12日、24日移动平均线的平均值）
   - 使用 bbi_deriv_uptrend 函数判断BBI是否整体上升（允许部分回撤）
   - 判断逻辑：在[20,60]个交易日窗口内，BBI归一化后的一阶差分满足一定条件：
   ···
   从近60日窗口开始统计，只要满足「下跌的天数比例不超过30%」就返回true，如果不满足就尝试近59日窗口，在不满足就尝试近58日窗口，以此类推，直到近20日窗口。如果还不满足就返回false。
   ···
      
2.KDJ指标过滤 （双重条件满足其一）
   - 当日J值小于阈值（配置为10）
   - 当日J值不高于最近60个交易日内J值的10%分位数
3.MACD指标判断
   - 计算DIF线（MACD的快线）
   - 要求当日DIF值大于0
4.价格波动约束
   - 最近60根K线内，收盘价的最大波动幅度不超过1倍（即(最高价/最低价-1)≤1）
从configs.json配置看，实际使用的参数比代码中的默认值更宽松：

- j_threshold=10（比默认的-5更宽松）
- bbi_min_window=20（比默认的90更小，更敏感）
- max_window=60（比默认的90更小）
- price_range_pct=1（表示允许100%的波动）
- bbi_q_threshold=0.3（允许30%的回撤，增加了策略的适应性）
这种组合设计旨在筛选出处于上升趋势但短期可能被低估的股票。


# bbi_deriv_uptrend 函数原理详解
该函数通过 归一化后的一阶差分分析来判断趋势，主要包含以下步骤：

1.归一化处理 ：将 BBI 序列按窗口起始值归一化，计算公式为：
BBI_norm(t) = BBI(t) / BBI(T-w+1)
其中 T 为最新数据点，w 为当前检测窗口长度。

2.一阶差分计算 ：计算归一化后的序列的一阶差分：
Δ(t) = BBI_norm(t) - BBI_norm(t-1)
正值表示上升，负值表示下降（回撤）。

3.分位数检验 ：检查一阶差分的前 q_threshold 分位数是否 ≥ 0。例如：
- q_threshold=0.1 表示允许 10% 的数据点出现下降
- q_threshold=0 表示要求全程单调不降（旧版行为）

4.窗口搜索策略 ：采用自最长窗口向下搜索的策略：
for w in range(longest, min_window - 1, -1):
    # 检查窗口是否满足条件

只要找到任一满足条件的区间，即返回 True。

## 参数说明
- bbi : pd.Series，BBI 序列（最新值在最后一位）
- min_window : 检测窗口的最小长度
- max_window : 检测窗口的最大长度；None 表示不设上限
- q_threshold : 允许一阶差分为负的比例（0 ≤ q_threshold ≤ 1）

## 函数特点
1.灵活的回撤容忍度 ：通过 q_threshold 参数控制允许的回撤程度
2.自适应窗口 ：从长到短尝试不同窗口长度，提高判断的适应性
3.归一化处理 ：消除绝对数值影响，专注于相对变化趋势
4.严格的参数校验 ：确保 q_threshold 在有效范围内 [0, 1]

该函数在选股策略中用于判断股票是否处于整体上升趋势，同时允许一定的短期回调，比单纯的单调上升判断更符合实际市场情况。
